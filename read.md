Forms & Tables
Recommended Time: ~2.5 hours

Phase: HTML – Building Web Foundations

Goal: Create a simple HTML form and a sample data table while ensuring basic accessibility.

Learning Materials
Odin Project – Form BasicsLinks to an external site.

Task
Open your Codespaces project.

Follow the Odin Project lesson to create a simple HTML form with fields for:

Name

Email

Message

Add a sample data table to your page.

Commit and push your changes.

Submit your GitHub repo link in Canvas.

Reflection
Ask ChatGPT:

Can you suggest two accessibility improvements for my form (e.g., labels, required fields, ARIA attributes) and explain why they matter?
Apply the suggested improvements and note them in your submission.
